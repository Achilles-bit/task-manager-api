# Task Manager API

A backend RESTful API for managing tasks, built using Node.js and Express.
The project demonstrates clean routing structure, HTTP method handling, and scalable API design principles.

\## Tech Stack

Node.js, Express, MongoDB



\## Endpoints

GET /api/tasks

POST /api/tasks

PUT /api/tasks/:id

DELETE /api/tasks/:id



\## Run

npm install

npm run dev
